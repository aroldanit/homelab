# Homelab — Alberto's DevOps Transition

A hands-on infrastructure lab built on bare-metal hardware, documenting my 
transition from IT Desktop Support to Cloud/Infrastructure Engineering.

## Goal
Build real, demonstrable skills in Linux, containers, cloud, and Kubernetes — 
documented publicly as a working portfolio.

## Target Stack
- **OS:** Ubuntu Server
- **Containers:** Docker, Docker Compose
- **Cloud:** AWS (EC2, S3, IAM, VPC, ECS)
- **IaC:** Terraform
- **Orchestration:** Kubernetes (K3s)
- **AI Infra:** Ollama + Nvidia Container Toolkit
